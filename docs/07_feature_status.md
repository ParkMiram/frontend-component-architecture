## 디바이스 목록

| 기능         | 상태 | 비고     |
|------------|----|--------|
| 디바이스 목록 조회 | ✅  |        |
| 디바이스 추가 팝업 | ⚠️ | UI만 구현 |
| 디바이스 수정 팝업 | ⚠️ | UI만 구현 |

### 현재 문제

#### 1️⃣ 발급장비 등록
- 등록 시 **500 Error 발생**

#### 2️⃣ IP 중복 처리
- 디바이스 추가 / 수정 시
- **IP 중복 정책 검토 필요**
- 백엔드와 협의 필요

#### 3️⃣ 디바이스 이름 생성

현재 임시 처리 dve_{num}
- 실제로는 **백엔드에서 생성 필요**
- 현재 **백엔드 미구현**
