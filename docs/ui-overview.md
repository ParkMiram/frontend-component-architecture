# UI Overview

아직 진행 중인 프로젝트이기 때문에, 데이터들은 모두 테스트 데이터입니다.

사용한 UI 컴포넌트 : `HeroUI` v2

HeroUI를 이용해 만든 커스텀 컴포넌트

- Modal (팝업) `src/organisms/BaseDrawer.tsx`
- Drawer (상세) `src/organisms/BaseModal.tsx`
- Table (데이터/페이지네이션) `src/organisms/DataTable.tsx`
- Search (검색) `src/molecules/Search.tsx`
- Form (입력 폼) `src/molecules/StyledForm.tsx`
- Button (버튼) `src/atoms/Button.tsx`
